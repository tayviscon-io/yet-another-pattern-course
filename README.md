# Курс по Паттернам Проектирования от [Tayviscon IO](https://github.com/tayviscon-io)
[![Official Project](https://img.shields.io/badge/Tayviscon_IO-official-%2363B175)](https://github.com/tayviscon-io/.github/blob/main/profile/README.md#tayviscon-io-github)

**Yet Another Pattern Course** - это курс от компании [Tayviscon IO](https://github.com/tayviscon-io)
из серии «[Yet Another Course](https://github.com/tayviscon-io/yet-another-course)»,
демонстрирующий пользу, классификацию и каталог паттернов проектирования,
которые вы можете использовать для решения часто встречающихся проблем при проектировании программ.
Стоит отметить, что вы можете вполне успешно работать, не зная ни одного паттерна, однако зная их,
вы получаете ещё один инструмент в свой личный набор профессионала.
